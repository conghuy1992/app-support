About Skyscraper Logic

Skyscraper Logic is a clean and modern brain puzzle built around pure logical deduction. Fill each row and column with exactly one of each unique item while leaving the remaining spaces empty.

Clues placed around the edges of the grid reveal which item appears first when viewed from that direction. Only the first visible item counts, making careful planning essential.

Enjoy infinite procedurally generated puzzles, complete offline gameplay, detailed gameplay recording, and multiple visual themes including Space, Magic, Pirate, Ice, Lava, and Jungle.
