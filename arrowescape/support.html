Arrow Escape: Arrow Out

A minimalist logic puzzle game where every arrow has only one way out. Clear the board by moving arrows in the correct order.

Available on the App Store and Google Play.

About Arrow Escape: Arrow Out

Arrow Escape: Arrow Out is a clean and challenging puzzle game built on a simple but clever rule: each arrow can only move in the direction it points.

An arrow can escape the board only when its path is completely clear. One wrong move may block other arrows, so planning and order matter. The game rewards logical thinking, patience, and foresight.

Arrow Escape features infinite puzzles with multiple difficulty levels including Easy, Medium, and Hard. The game works fully offline and supports both Light Mode and Dark Mode for comfortable play anytime.
